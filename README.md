# Age Prediction Project

A Machine Learning based project that predicts age from facial images using Deep Learning and Computer Vision.

## Features

* Face Detection
* Age Prediction
* Real-time image processing
* Deep learning model training

## Technologies Used

* Python
* OpenCV
* TensorFlow / Keras
* NumPy
* Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```
